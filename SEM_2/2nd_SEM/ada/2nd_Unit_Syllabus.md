# SEARCHING TECHNIQUES
- Linear Search =>(Completed) =>**BestCase=OMEGA(1) ; WORST CASE=O(n)**
- Binary Search(Recursive and non recursive both) =>(Completed) **Worst_Case=O(logn) ; BESTCASE=OMEGA(1)**
- Interpolation Search **Formul =low+((target-arr[low])*high-low)/arr[high]-arr[low] **  BEST CASE:- omega(1) Worst Case:- O(n)
- Exhaustive Search **Always exponential in nature**.
