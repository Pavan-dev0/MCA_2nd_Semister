# Syllabus: Computer Networks

## Course Overview
This course introduces the fundamental concepts of computer networking, covering both theoretical foundations and practical applications.

## Topics

### 1. Introduction
- Overview of the Internet
- What is the Internet?

### 2. The Network Edge
- End systems, clients, servers
- Access networks and physical media

### 3. The Network Core
- Packet switching vs. circuit switching
- Network structure

### 4. Performance Metrics
- Delay
- Loss
- Throughput in packet-switched networks

### 5. Protocol Layers & Service Models
- Layered architecture
- OSI and TCP/IP models
- Service abstraction

### 6. Principles of Network Applications
- Design considerations
- Application-layer protocols

### 7. The Web and HTTP
- Web architecture
- HTTP request/response cycle

### 8. Electronic Mail
- SMTP basics
- Email architecture

### 9. Domain Name System (DNS)
- DNS as the Internet’s directory service
- Name resolution process

---

## Experiential Learning

### Packet Tracer Tool
- Getting started with Packet Tracer
- Creating networks using hubs, switches, and routers

### IP Addressing
- Overview of IP addressing
- Subnetting basics

### Routing
- Creating static routing tables in routers
- Understanding routing decisions

### TCP/IP Layering
- Layering principles in TCP/IP networks
- Encapsulation and decapsulation

### Protocol Communication
- Understanding HTTP communication
- Understanding DNS communication

---

## Learning Outcomes
By the end of this course, students will be able to:
- Explain the structure and functioning of the Internet.
- Analyze performance metrics in packet-switched networks.
- Apply protocol layering concepts to real-world scenarios.
- Configure basic networks using Packet Tracer.
- Implement static routing and understand IP addressing.
- Demonstrate communication using HTTP and DNS protocols.
