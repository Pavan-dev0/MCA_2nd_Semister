Subject 01 · Core

Problem Solving &
Python Programming

Algorithms, data types, control flow, functions and object-oriented design in Python.

4 Credits

An introduction to algorithmic thinking and structured programming using Python. Students learn to design efficient solutions to computational problems through hands-on coding exercises covering the full language core.

Chapters

CH 01
Introduction to Computing
Algorithms, flowcharts, pseudocode
CH 02
Python Fundamentals
Variables, types, operators, I/O
CH 03
Control Structures
if, for, while, match-case
CH 04
Functions & Modules
Scope, recursion, exception handling
Subject 02 · Core

Data Structures
& Algorithms

Linear and non-linear structures, complexity analysis and algorithm design strategies.

4 Credits

Comprehensive study of data organisation and manipulation strategies. Covers time-space complexity, classic sorting and searching algorithms, and recursive problem solving — the foundation of competitive programming.

Chapters

CH 01
Arrays & Linked Lists
Singly, doubly, circular
CH 02
Stacks, Queues & Deques
Applications and implementations
CH 03
Trees & Binary Search
BST, AVL, Heap, Trie
CH 04
Graphs & Sorting
BFS, DFS, QuickSort, MergeSort
Subject 03 · Core

Database Management
Systems

Relational model, SQL, normalisation, transactions and query optimisation.

4 Credits

Covers the theory and practice of organising, storing and retrieving data efficiently. Students gain deep proficiency in SQL and learn to design robust, normalised schemas for real-world applications.

Chapters

CH 01
ER Model & Relational Model
Entities, relationships, keys
CH 02
SQL — DDL, DML, DCL
Joins, subqueries, views
CH 03
Normalisation
1NF → 2NF → 3NF → BCNF
CH 04
Transactions & Recovery
ACID, concurrency control
Subject 04 · Core

Computer
Networks

Network architecture, layered protocols and internet communication fundamentals.

4 Credits

Study of data communication and computer networks from the physical layer up to the application layer. Special focus on TCP/IP, routing, security principles and emerging network paradigms.

Chapters

CH 01
OSI & TCP/IP Models
Layer functions and interactions
CH 02
Data Link Layer
Framing, MAC, error control
CH 03
Network & Transport Layer
IP addressing, TCP vs UDP
CH 04
Application Protocols
HTTP, DNS, SMTP, FTP, SSH
Subject 05 · Core

Discrete Mathematics
& Logic

Mathematical foundations essential for computing — logic, sets, graphs and combinatorics.

4 Credits

Equips students with the mathematical tools underlying computer science: formal logic for reasoning about programs, combinatorics for algorithm analysis, and graph theory for network modelling.

Chapters

CH 01
Propositional Logic
Truth tables, tautologies, proofs
CH 02
Set Theory & Relations
Functions, equivalence, partial orders
CH 03
Combinatorics
Permutations, combinations, pigeonhole
CH 04
Graph Theory
Paths, trees, Euler & Hamiltonian
