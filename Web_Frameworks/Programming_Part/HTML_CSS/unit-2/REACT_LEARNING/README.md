# REACT
- React is the javascript library thats used to build the web applications
- react focuses on building web application using the components(Just like the building blocks)
- React uses the syntax extention of the javascript known as the jsx known as the javascript XML(writes the html like code in the javascript files)
- React also utilizes the virtual DOM (a lightweight version of the real DOM of the webpage) we can keep track when the changes met to the virtual dom and only apply that specific change to the real DOM without needing to refresh the entire webpage. This reduces the rendering performance overhead

- Note that the App.jsx is also the component
